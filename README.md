# Student Management Dashboard (React)

A responsive **Student CRUD Dashboard** built with React.
Users can add, edit, delete, and search students with a modern drawer-based UI.

---

## Live Demo

https://student-management-9754ab.netlify.app/

---

## Features

* Add new students
* Edit student information
* Delete students
* Search students instantly
* Responsive UI for mobile and desktop
* Drawer-based form interface
* Student statistics dashboard
* LocalStorage data persistence

---

## Tech Stack

* React
* JavaScript (ES6+)
* CSS
* Remix Icons

---

## Installation

Clone the repository:

git clone https://github.com/jawadgit23/Student-CRUD-App

Go to the project folder:

cd student-management-dashboard

Install dependencies:

npm install

Run the project:

npm run dev
